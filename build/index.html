Exercise Tracker

Create a new user

POST /api/exercise/new-user



Add an exercise

POST /api/exercise/add









Get Exercise Logs

GET /api/exercise/log









Use either form or GET route from browser

userId is required, all other fields are optional

GET users's exercise log: GET /api/exercise/log?userId=number&from=date&to=date&limit=number

from, to = dates (yyyy-mm-dd); userId = number, limit = number
